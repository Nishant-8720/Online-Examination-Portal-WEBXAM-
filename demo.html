INTRODUCTION

WEBXAM is a one type of Online Examination System contains providers to focus on Competitive MCQ Exam System. In the System we present techniques that are pertinent to the elements of assessment process: answers submission, computerized grading, and feedback after submission.

As the modern organizations are automated and computers are working as per the instructions, it becomes essential for the coordination of human beings, commodity and computers in a modern organization.

The administrators, instructor, Students who are attending for online examination can communicate with the system through this project, thus facilitating effective implementation and monitoring of various activities of Online Examinations like conducting Exams as per scheduled basis and delivering result to that particular use or student. And the details of students who attempted Online Examination are maintained at administrator.

In this project administrators has rights to add a new MCQ Quiz and to remove a Quiz. And admin can also define the quiz time duration or its set the marking scheme depends on the question. The project is also has a one feature of negative marking scheme. The admin can also see the all the user information who attempt the quiz or it has also display the result and ranking depends on user result. In this project the admin has special user name and password that is use to see the admin dashboard. Admin has separate dashboard to see or manage the user activity.

In that the user side that has a separate dashboard that content the Exam Portal and user also can see the raking list. Or after completing the test the result will be automatically generated in term of Total questions, Right answer, Wrong answer, Score, Overall Score. And the user can also see the history and ranking. In this project is user has one more feature to search quiz.
